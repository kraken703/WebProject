Basic html Tags


html> - Defines the beginning and end of an HTML document.
head> - Contains metadata such as the title, styles, and scripts that are not displayed on the webpage.
title> - Defines the title of the webpage that appears in the browser tab.
body> - Contains the visible content of the webpage.
h1>-h6> - Defines headings with different levels of importance.
p> - Defines a paragraph of text.
a> - Defines a hyperlink that links to another webpage or a specific location on the same page.
img> - Displays an image on the webpage.
ul> - Defines an unordered list.
ol> - Defines an ordered list.
li> - Defines a list item.
div> - Defines a container element that groups other elements together.
span> - Defines a container element that groups inline elements together.
form> - Defines a form for user input.
